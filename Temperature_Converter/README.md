# 🌡️ Temperature Converter

A simple Python script to convert temperatures between Celsius, Fahrenheit, and Kelvin.

## 📁 File

- `temperature_converter.py` – Main script that performs temperature conversions.

## 🚀 Features

- Convert:
  - Celsius ↔ Fahrenheit
  - Celsius ↔ Kelvin
  - Fahrenheit ↔ Kelvin
- Input validation and error handling

## 🧪 Usage

```bash
python temperature_converter.py
```

You'll be prompted to:
1. Enter the temperature value
2. Choose the source unit (Celsius, Fahrenheit, or Kelvin)
3. Choose the target unit

Example:
```bash
Enter temperature: 100
From (C/F/K): C
To (C/F/K): F
Result: 212.0°F
```

## 🛠️ Requirements

- Python 3.x

No external libraries are required.
